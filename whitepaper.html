USDTs Token contract (USDTs) – Official Whitepaper

Decentralized, Multi-Collateral Stablecoin on Ethereum | Version 1.0 – April 2025

1. Abstract

USDTs Token contract (USDTs) is a next-generation, multi-collateral stablecoin built on Ethereum, designed to maintain a stable 1:1 peg to the US Dollar. It combines transparent reserve architecture, on-chain governance, and upgradeable smart contracts to ensure long-term resilience.

3. Token Mechanics

Minting & Burning

Only authorized minters (multisig) can mint USDTs Token contract. Burning is permissionless. Daily mint limit: 1% of supply.

Governance

1 USDTs = 1 vote. Quorum: 5%. Voting period: 72 hours.

Vote on Snapshot

Fee Distribution

  • 70% → Reserve Fund
  • 20% → Stakers
  • 10% → Treasury

4. Reserve Architecture

The protocol uses a phased collateral model:

  • Phase 1: 1:1 USDC
  • Phase 2: 1.5:1 (USDC + stETH)
  • Phase 3: 2:1 Dynamic

5. Governance

USDTs Token contract holders govern the protocol via Snapshot. Proposal types include reserve changes, fee adjustments, and upgrades.

7. Smart Contract

Built with OpenZeppelin's upgradeable libraries. Uses UUPS proxy for secure upgrades.

function mint(address to, uint256 amount) 
    public onlyRole(MINTER_ROLE) nonReentrant 
{
    require(to != address(0), "Mint to zero address");
    _mint(to, amount);
}

function burn(address from, uint256 amount) 
    public onlyRole(BURNER_ROLE) nonReentrant 
{
    require(from != address(0), "Burn from zero address");
    _burn(from, amount);
}
